📋 1. 개인정보의 수집 및 이용

Fragment of Mystery (단서의 파편)는 사용자의 개인정보를 수집하거나 처리하지 않습니다.

  • ❌ 이름, 이메일, 전화번호 등 개인 식별 정보 수집 안함
  • ❌ 위치 정보, GPS 데이터 수집 안함
  • ❌ 기기 고유 식별자 수집 안함
  • ❌ 사용 패턴, 분석 데이터 수집 안함
  • ❌ 쿠키, 추적 기술 사용 안함
  • ❌ 카메라, 마이크, 연락처 접근 안함

💾 2. 게임 데이터 저장

게임 진행 상황 및 설정은 사용자의 기기에만 로컬로 저장되며, 외부 서버로 전송되지 않습니다.

  • ✅ 게임 진행 상황은 기기 내부 저장소에만 저장
  • ✅ 서버로 데이터 전송 없음
  • ✅ 클라우드 저장 기능 없음
  • ✅ 제3자와 정보 공유 없음
  • ✅ 앱 삭제 시 모든 데이터 완전 삭제

🌐 3. 인터넷 연결 및 통신

본 앱은 완전한 오프라인 게임으로, 인터넷 연결이 필요하지 않습니다.

  • ✅ 100% 오프라인 작동
  • ✅ 외부 서버와 통신 안함
  • ✅ 네트워크 권한 사용 안함
  • ✅ 데이터 사용량 0MB

📢 5. 광고 및 제3자 서비스

본 앱에는 광고나 제3자 추적 서비스가 포함되어 있지 않습니다.

  • ❌ 광고 SDK 사용 안함
  • ❌ 분석 도구 사용 안함

👶 6. 어린이 개인정보 보호

본 앱은 13세 이상을 대상으로 하며, 13세 미만 어린이의 개인정보를 의도적으로 수집하지 않습니다.

  • 🎯 타겟 연령: 13세 이상
  • ✅ COPPA (어린이 온라인 개인정보 보호법) 준수
  • ❌ 어린이 대상 데이터 수집 안함

🔒 7. 데이터 보안

개인정보를 수집하지 않으므로 데이터 유출 위험이 없습니다.

  • ✅ 개인정보 수집 안함 = 100% 안전
  • ✅ 해킹, 데이터 유출 위험 없음
  • ✅ 암호화 필요 없음 (데이터 수집 안함)
